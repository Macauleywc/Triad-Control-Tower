Connect your OneDrive Excel file — two ways:
Option A — Sign in with Microsoft (recommended)
  1. Configure your Azure App ID in the script (see SETUP.md)
  2. Click Sign in with Microsoft above
  3. Click Choose file and pick your Excel workbook from OneDrive

  1. In Excel: File → Save As → CSV (Comma delimited)

Import template — triad_delivery_template.csv
Fill in one row per route area · save as CSV · drop above
load_ref delivery_name * delivery_postcode * pallets delivery_time
* Required fields.  ·  load_ref — vehicle or run reference (e.g. "RUN1", "VAN2"). Stops sharing a load_ref are on the same vehicle.  ·  delivery_name — customer name or location.  ·  delivery_postcode — full or partial postcode.  ·  pallets — pallets for this stop.  ·  delivery_time — estimated arrival HH:MM.  ·  One row per delivery stop · one file per site per day.
